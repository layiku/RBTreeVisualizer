# RBTree Visualizer - 技术规格说明书 (Spec)

[中文](spec_zh.md) | [English](spec_en.md)

## 1. 项目概述
一个高保真、基于 Web 的红黑树 (RBT) 操作可视化教学工具。该工具专注于“预告-执行”心理模型，帮助学生理解树平衡的因果关系。

## 2. UI 与 UX 架构

### 2.1 布局拆解
- **侧边栏 (左侧, 380px)**：
    - **Header**：包含标题、副标题和 **中英文切换按钮**。
    - **控制组**：数值输入（1-99）和“插入”按钮。
    - **速度控制**：支持 0.1x 到 2.0x 的倍速调节，带实时数值显示。
    - **按钮组**：
        - 第一排：[上一步] [下一步] [自动播放]。
        - 第二排：[完成本次平衡] [重置整棵树]。
    - **规则指南**：包含红黑树 4 大核心特性的卡片，支持长句自动换行。
- **主画布 (中心右侧)**：Canvas 渲染区域，带点阵背景。
- **状态日志 (底部, 100px)**：全宽、单行日志，用于显示分步算法逻辑，支持毛玻璃特效。

## 3. 算法规范 (`js/rbtree.js`)

### 3.1 数据模型
- `RBNode`：包含 `value`, `color`, `left`, `right`, `parent`。
- `RBTree`：维护根节点指针和 `steps` 状态历史数组。

### 3.2 状态捕获 (快照机制)
每次插入都会触发一系列快照。每个快照对象必须包含：
- `root`：深拷贝的树结构。
- `descKey`：指向 `i18n.js` 的翻译键名。
- `params`：用于翻译模板的动态参数（如当前节点数值）。
- `activeNodeValue`：当前操作节点的数值（用于视觉高亮）。

## 4. 可视化与动画 (`js/visualizer.js`)

### 4.1 渲染引擎
- **Canvas 2D API**：支持 High-DPI (`devicePixelRatio`)。
- **节点表现**：使用 **径向渐变** 模拟球体质感，带物理投影。

### 4.2 动效设计
- **缓动函数**：使用 `Ease-In-Out Cubic` 实现非线性的物理位移。
- **颜色过渡**：使用 **RGB 插值 (Lerp)** 实现丝滑的变色效果。
- **入场动画**：新节点从父节点位置平滑滑入，或从上方落下。

## 5. 交互状态机 (`js/main.js`)

### 5.1 国际化 (I18n)
- 集中式字典管理 (`js/i18n.js`)。
- 运行时切换语言，实时更新所有 UI 标签和已存在的算法日志。

### 5.2 无限回滚逻辑
- 点击“上一步”可跨越插入操作，支持一直回退到初始空树状态。

## 6. 设计规范
- **字体**：UI 使用 Inter，日志使用 JetBrains Mono。
- **视觉风格**：极简主义，大量使用 `backdrop-filter` 磨砂玻璃特效。

# 红黑树分步可视化教学工具 (RBTree Visualizer)

[English](README.md) | [中文](README_zh.md)

这是一个为计算机科学 education 设计的红黑树算法演示工具。它通过极高的视觉保真度和创新的“预告-执行”分步逻辑，让红黑树的自平衡过程变得前所未有的清晰。

## 🌟 核心特性

- **教学级步进逻辑**：不仅仅是展示结果，而是在旋转和变色发生前先给出详细的逻辑预告，引导学习者思考。
- **物理级平滑动画**：采用 **Ease-In-Out Cubic** 缓动函数，使节点在旋转和重排时具有丝滑的位移轨迹。
- **焦点高亮系统**：活跃节点拥有动态呼吸脉冲光环，确保学习者的注意力始终锁定在当前操作点。
- **无限历史回滚**：支持跨操作的撤销功能，可以一路回退到空树状态，非常适合对比不同插入顺序的结果。
- **自适应响应式布局**：采用通栏算法日志设计，自动适配不同屏幕尺寸，确保信息阅读的高效与舒适。
- **国际化多语言**：支持中英文一键切换，实时翻译所有算法日志描述。
- **极致美学**：基于 Inter 字体和毛玻璃特效，提供极简主义且富有质感的现代 UI。

## 🏗️ 代码架构与技术实现

- **核心算法层** (`rbtree.js`)：纯 JavaScript 实现的红黑树，内置专为教学设计的“快照机制”，支持捕获算法中间状态。
- **视觉引擎层** (`visualizer.js`)：高性能 Canvas 2D 渲染引擎，支持径向渐变质感、基于 Lerp 的平滑动画及自适应缩放。
- **国际化引擎** (`i18n.js`)：集中式翻译字典，支持 UI 文字及动态生成的算法日志实时中英文切换。
- **逻辑控制层** (`main.js`)：基于状态机的逻辑协调器，负责管理用户交互、历史记录堆栈及动画播放流。

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`。
2. 在左侧面板输入 **1-99** 之间的数字并点击“插入”。
3. 使用 **[上一步]**、**[下一步]** 或 **[自动播放]** 控制演示节奏。
4. 观察页面底部 **通览日志** 中的详细算法描述。
